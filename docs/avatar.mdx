```jsx
import { Avatar } from "fundamental-react-native";
```

An **Avatar** is a visual presentation option around using an icon or user initials.

## Examples

### Primary

```SnackPlayer name=Button&dependencies=fundamental-react-native@0.0.3,@expo/vector-icons
import React from 'react';
import { View } from 'react-native';
import { Avatar, ThemeProvider, theme } from 'fundamental-react-native';

const App = () => (
  <ThemeProvider value={theme}>
    <View style={{
      flex: 1,
      justifyContent: 'center',
      alignItems: 'center',
    }}>
      <Avatar iconSet="FontAwesome" glyph="user" size="m" />
    </View>
  </ThemeProvider>
);

export default App;
```
