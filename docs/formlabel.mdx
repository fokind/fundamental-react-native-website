```jsx
import { FormLabel } from "fundamental-react-native";
```

A **FormLabel** is used to identify form components such as **FormInput**, **Checkbox** and **TextArea**. Best practice is to use this component as a child of **FormGroup**.

## Examples

### Primary

```SnackPlayer name=Button&dependencies=fundamental-react-native@0.0.3
import React from 'react';
import { View } from 'react-native';
import { FormInput, FormItem, FormLabel, ThemeProvider, theme } from 'fundamental-react-native';

const App = () => (
  <ThemeProvider value={theme}>
    <View style={{
      flex: 1,
      justifyContent: 'center',
      alignItems: 'center',
    }}>
      <FormItem>
        <FormLabel>Default Label</FormLabel>
        <FormInput />
      </FormItem>
    </View>
  </ThemeProvider>
);

export default App;
```
